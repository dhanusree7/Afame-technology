# Afame Technologies Data Analysis Projects

This repository contains two data analysis projects completed as part of an internship at Afame Technologies.

## Titanic Survival Prediction
The goal of this project is to build a model that predicts whether a passenger on the Titanic survived or not based on various features.

### Code
- The code for this project is located in the `titanic/code` directory.

## Sales Data Analysis
The goal of this project is to analyze sales data to find patterns, best-selling items, and revenue indicators to help in business decision-making.

### Code
- The code for this project is located in the `sales/code` directory.
